# yii2 install documentation

## Инструкция для исползования

### Клонирование проекта и копирование env

```bash
$ git clone https://github.com/JaloladdinRozmetov/get-currency.git
$ cd get-currency
$ cp .env.example .env
```

### 
```bash
$ composer install
```
set database in main.php

### and run  
```bash
$ yii migrate
```
